```Thread Budget Formula computed startup CPU core count:```
if cpu_count < 3 → server refuses to start

fixed:
  event_loop   = 1
  writer       = 1
  checkpoint   = 1
  presence     = 1

variable:
  remaining    = max(cpu_count, 4) - 4
  readers      = min(4, max(1, remaining / 2))
  notification = max(1, remaining - readers)
